# 城市地标

## 模板

### 城市微缩景观

```
一张充满[风格]的[城市名]地图/景观，[宽高比]。

[主要创意描述，如"地图上'生长'出一座栩栩如生的城市微缩立体模型"]

核心地标：
- [地标1：详细描述外观和特征]
- [地标2：详细描述]
- [地标3：详细描述]

动态元素：
- [如"微型无人机编队在空中闪烁飞行"]
- [如"地铁列车从隧道口驶出"]

光线：[如"工作室柔光照明，电影质感的光影，画面深邃"]

风格指令：[如"3D微缩景观风格"]

画质：超详细
```

> **技巧**："地图上生长出立体模型"是最核心的创意指令。地标名称越具体效果越好。换成其他城市只需改地标名称。

## 示例

### 深圳微缩景观

> 一张充满街头风格的深圳地图，平铺在光滑反光的深色表面上，地图上"生长"出一座栩栩如生的深圳微缩立体模型：平安金融中心、华润总部"春笋"、万象城购物中心清晰可见，梧桐山上绿植葱茏，华强北电子市场摊位前人群熙攘，深圳地铁列车从隧道口驶出，微型无人机编队在梧桐山上空闪烁飞行。所有元素都从地图表面拔地而起，构成一座3D城市微缩景观。工作室柔光照明，电影质感的光影，画面深邃，采用2：3画幅比例。
