# 信息图/食谱

## 模板

### 通用信息图

```
一张[风格]的[主题]信息图，[宽高比]。

主体内容：[主要视觉元素的描述]

信息布局：
- 标题区域：[标题文字]，位于[位置]
- 主体区域：[核心信息的视觉呈现方式]
- 说明区域：[补充信息的位置和呈现方式]

配色方案：[颜色描述]

排版风格：[如"动态的编辑式布局"、"整洁的网格布局"]

文字内容：
- [需要渲染的具体文字列表]

风格指令：[如"干净现代的信息图风格"、"扁平化设计"]
```

### 食谱信息图

```
一张干净现代的食谱信息图。以视觉诱人的成品形式展示【菜名】，可以是切片、装盘或分好份的状态，食材略微悬浮，采用透视或倾斜视角。

配料部分：为每种配料配上带数量的小图标或迷你插图，以簇状、列表或循环流动的形式排列，并在视觉上与主菜品相连。
步骤部分：用带编号的框格、箭头或线条展示准备步骤，在主菜周围形成一个逻辑流程。可酌情加入小的烹饪图标（刀、锅、烤箱、计时器）。
```

> **技巧**：`【菜名】` 替换为实际菜名。食材略微悬浮 + 透视视角让信息图更有立体感。

## 示例

### 决策信息图（自动风格）

> I want to wash my car. The car wash is 50 meters away. Should I walk or drive?

> **技巧**：AI 图像生成模型会自动将这类问题转化为信息图。不需要额外指定"画成信息图"。
