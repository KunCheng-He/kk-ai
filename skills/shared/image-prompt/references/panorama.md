# 全景长卷

## 模板

```
[画面内容的总体概述]，[宽高比如 21:9 或 8:1] 的超宽全景图。

画面从左到右依次展开：
[左侧区域描述]
→ [中左区域描述]
→ [中间区域描述]
→ [中右区域描述]
→ [右侧区域描述]

画面细节：
- 近景：[近处元素描述]
- 中景：[中景元素描述]
- 远景：[远景元素描述]
- 过渡：[区域之间的过渡描述]

氛围：[整体氛围]

画质：超详细，高分辨率
```

> **技巧**：
> - 指定宽高比是生成全景图的关键。21:9 适合横幅，8:1 适合超长卷。
> - 全景图需要**从上到下或从左到右的层次描述**，用箭头或分号分隔不同区域。
> - "极小的[人物]"和"巨大的[物体]"形成比例对比。
> - 需要更高清时在末尾加上"4K 分辨率"。

### 万米深潜

> 万米深潜。画面构想：这是一场向海洋极深处的坠落。最上方是波光粼粼的海面和一艘小船；往下是游动着巨大蓝鲸；继续往下光线急剧变暗，出现沉船和发光水母；到了画面的最底部，是一个几乎占据整个屏幕宽度的、潜伏在海沟里的不可名状的克苏鲁巨兽张开的深渊巨口，而上方正有一个极小的潜水员在缓缓下落。8:1 超长图。
