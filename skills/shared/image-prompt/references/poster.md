# 海报/信息图

## 模板

```
一张[风格类型]海报，[宽高比]。

主题：[核心主题/标题]

主视觉：[画面的核心视觉元素描述，包括人物/物体/场景的详细外观]

排版布局：
- 顶部区域：[标题文字内容]，[字体风格描述]
- 中部区域：[核心视觉元素位置和描述]
- 底部区域：[副标题/日期/补充信息]，[字体风格描述]

色彩方案：[主色调和辅色调描述]

风格指令：[整体风格关键词]

文字渲染要求：[需要渲染的具体文字内容、位置、字体大小关系]

画质：超详细，高清，专业排版设计
```

### 全球化海报翻译变体

```
Translate all text in this [image/poster/advertisement] to the language of [target language]. ONLY translate the text - do not add any cultural imagery, flags, national symbols, or stereotypical visual elements. Keep the image, composition, styling, colors, and all visual elements exactly the same as the original. The only change should be the language of the text.
```

> 全球化翻译提示词模板。建议用英文提示词。

## 示例

### 电影海报

> 生成一张完整的电影海报，包含中英文标题、角色介绍、经典台词、上映信息等，排版专业，文字清晰。星际穿越

### 古诗配图

> 根据以下古诗画一幅画,并附上原文。[诗句内容]

画面呈现[朝代][场景描述]，诗句清晰地印在画面上，字体优美。

> **技巧**：需要渲染中文文字时，"附上原文"是最简洁的指令。

### 全球化翻译

> Translate all text in this advertisement image to the language of Japanese. ONLY translate the text - do not add any cultural imagery, flags, national symbols, or stereotypical visual elements. Keep the image, composition, styling, colors, and all visual elements exactly the same as the original. The only change should be the language of the text.

> **技巧**：用英文提示词效果更好，中文版可能导致文字渲染下降。
