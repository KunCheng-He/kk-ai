# 动漫风格

## 模板

### 2D 动漫风格

```
[动漫作者/风格名]风格，[宽高比]。

场景：[场景描述，需要特别注明光影效果和氛围]

人物：
- 外貌：[人物外貌描述]
- 服装：[服装描述]
- 动作：[人物动作和姿态]
- 表情：[表情描述]

光影效果：[详细的光影描述]

色彩：[色彩方案，如"色彩清澈如宝石"]

氛围：[情感氛围描述，如"梦幻细腻，情感氛围浓郁"]

风格指令：动漫壁纸风格，超高饱和度光影
```

> **关键光影词**：光粒感、水坑倒映、玻璃幕墙反射、夕阳光束、尘埃光点

### 3D 动画风格

```
[角色列表，逐个描述]，生成群体自拍照/场景，每个角色都有独特的表情和姿态，风格统一且细节丰富。3D动画电影风格，Pixar/迪士尼风格
```

## 示例

### 新海诚风格

> 新海诚风格，超高饱和度光影，雨后街角，一位身穿透明雨衣的女孩站在画面中央，地面水坑倒映着深蓝色天空和散落的金色光芒，背景是被雨水冲洗干净的城市建筑，玻璃幕墙反射着暖色调夕阳，光束交织，空气中闪烁着微小的尘埃光点，整个场景充满"光粒感"，女孩发丝被微风轻轻吹起，色彩清澈如宝石，画面整体梦幻细腻，情感氛围浓郁，动漫壁纸风格。

> **技巧**：新海诚风格的关键词是**光粒感**、**超高饱和度光影**、**水坑倒映**、**玻璃幕墙反射**。

### 可爱四宫格

> 传照片人物形象还原，精准保留发型、面部细节和表情，明亮清新青春时尚人像，纵向拼贴画格式。她穿着时尚年轻服装，手持红色棒棒糖靠近嘴唇，脸上装饰着可爱的彩色贴纸，心形、草莓和星星形状，营造活泼潮流氛围，眼睛富有表现力，妆容柔和自然，表情梦幻。背景是贴满彩色便利贴和备忘录的软木板，营造校园和手帐风格，手绘涂鸦星星、箭头等元素增添可爱杂志美学，主人物周围环绕着约4张不同形状的剪贴簿风格相框，展示同一个人的不同服装和姿势。整体图像采用柔和影棚灯光，高清细节，清新剪贴簿美学，韩系时尚画报风格，色彩鲜艳。

> **技巧**："纵向拼贴画格式"指定排版。"剪贴簿美学"带出手工感。
