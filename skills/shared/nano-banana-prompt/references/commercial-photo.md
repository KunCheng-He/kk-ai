# 商业摄影

## 模板

```
超高端商业摄影，[宽高比]。

主角：[产品/人物的详细描述]
- 形状：[产品形状描述]
- 材质：[材质、纹理、光泽]
- 品牌标识：[需要出现的品牌文字和位置]
- 颜色：[具体颜色描述]

场景布置：
- 主体位置：[居中/偏移，角度描述]
- 表面：[放置面的材质和颜色]
- 倒影：[是否有镜像倒影，描述倒影效果]
- 周围元素：[环绕的产品搭配物描述，对称/不对称构图]

背景：[背景描述，如"平滑的暖色调渐变，从左边的淡黄色过渡到右边的深橙色"]

光线：[光线描述，如"强烈的柔光摄影棚主光源从左上角照射"]

色彩方案：[主色调列举]

纹理细节：[需要强调的纹理细节]

景深：[景深描述]

风格指令：高端商业[行业]广告风格，简洁现代构图，电影级真实感，超详细8K照片级真实，影棚专业品质
```

> **核心**：商业摄影提示词的关键是材质和纹理的超细描述（玻璃反光、液体粘稠感、果肉质感），以及光线方向的具体指定。

## 示例

### 饮料产品广告

> 超高端商业摄影，主角为统一鲜橙多橙汁：一个透明圆柱形玻璃瓶，瓶身优雅地居中略微后倾，装满鲜亮橙色、带有可见果肉的橙汁。瓶身正面标签清晰可见"统一鲜橙多"品牌标识和橙子图案，配以哑光白色瓶盖和金属质感橙色环。瓶子放置于光泽感极强的橙色反光液体表面上，下方形成清晰的镜像倒影。周围环绕着完整和切开的柑橘类水果，包括鲜艳的橙子、切半露出饱满果肉的红肉西柚、以及切片黄柠檬，围绕瓶子形成对称构图。新鲜的小橙汁滴和气泡在瓶底周围空中凝固，新鲜的绿色柑橘叶轻柔地漂浮在背景中。背景为平滑的暖色调渐变，从左边的淡黄色过渡到右边的深橙色。采用强烈的柔光摄影棚主光源从左上角照射，在玻璃和液体上形成清晰的高光，橙汁散发出明亮的光泽，有细微的阴影和高对比度反光。色彩以鲜艳的橙色、黄色、金色和清新的绿色点缀为主。纹理极其细致，包括光滑的玻璃、粘稠的液体质感、柑橘皮孔、多汁的果肉和反光表面。景深控制精准，瓶身标签和前景水果锐利清晰，背景呈现柔和的虚化效果。高端商业饮料广告风格，简洁现代构图，电影级真实感，超详细8K照片级真实，影棚专业品质。
