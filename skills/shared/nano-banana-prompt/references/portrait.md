# 人物写真

## 模板

```
超写实摄影，电影质感，[宽高比]。

人物描述：
- 外貌：[面部特征、表情描述]
- 服装：[服装的详细描述，包括材质、颜色、纹理、装饰]
- 发型：[发型和发饰的详细描述]
- 配饰：[手持物品、佩戴饰品的详细描述]
- 姿势：[身体姿态描述，如"3/4 侧背影回眸"]

场景：[场景描述，如"故宫雪夜"]

氛围元素：
- 前景：[前景元素，如"失焦的飘带与雪花"]
- 背景：[背景元素描述]
- 光影：[光影效果描述]
- 特殊效果：[飘动、粒子、光晕等]

构图：[构图指令]

色彩：[色彩方案，如"富士胶片风，主调绛红、鎏金、赤橘，对比白雪与黑夜"]

氛围：[整体氛围描述]

画质：8K，超高细节
```

> **技巧**：
> - 用摄影参数（85mm、f/1.2）控制画面感
> - "前景有失焦的[元素]"创造层次感
> - 色彩方案用"主调X、Y、Z，对比A与B"的格式更精确

## 示例

### 古风人像

> 超写实摄影，电影质感，85mm焦距，f/1.2大光圈浅景深，特写。人物面部与服装严格基于用户上传的参考图。一位女子头戴巨大金色凤冠与繁复发饰，身着华丽传统服饰，以3/4侧背影回眸，表情空灵。手中橙色透明丝绸飘带向镜头飘动。场景为故宫雪夜，发光建筑提供强烈逆光形成轮廓光晕，正面补光确保面部清晰。构图刻意使凤冠与故宫飞檐在视觉上呼应。前景有失焦的飘带与雪花。色彩为富士胶片风，主调绛红、鎏金、赤橘，对比白雪与黑夜。氛围雍容神性，如故宫上元神话。8K，超高细节。
