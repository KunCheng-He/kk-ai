# 故事板/连续画面

## 模板

### 连续故事板

```
生成[数量]张连续的[故事/主题]画面，[宽高比]。

每张画面需保持以下一致性：
- 角色一致性：[每个角色的固定外观特征描述]
- 风格一致性：[统一的视觉风格]

画面序列：
第1张：[场景描述] - [角色动作] - [对白/文字]
第2张：[场景描述] - [角色动作] - [对白/文字]
...
第N张：[场景描述] - [角色动作] - [对白/文字]

对白要求：[语言] 对白，[字体风格要求]

风格指令：[整体风格]
```

> **技巧**：Nano Banana 2 支持最多 5 个角色的一致性。在首次生成时详细描述每个角色的固定特征，后续生成复用相同描述。

### 九宫格/多宫格变体

```
生成[人物名]的九宫格大头贴照片，背景是在拍大头贴的房间内，一只手拿着这张九宫格照片，每个宫格都是不同的动作和表情，有[情绪1]也有[情绪2]，凸显出[性格特点]
```

## 示例

### 三打白骨精故事板

> 生成一个西游记三打白骨精的故事，一共9张图片，我提供三个分镜图片供参考，图片使用中文对白：
> 第1张：[场景1描述] - [角色动作] - [对白]
> ...
> 生成9张连续的故事图，每张图里的唐僧、孙悟空、猪八戒、沙僧形象保持一致，对白清晰，剧情连贯。

> **技巧**：上传参考图可以保持角色一致性。可以指定风格变换，如"儿童绘本风格"。

### 多宫格大头贴

> 生成马斯克的九宫格大头贴照片，背景是在拍大头贴的房间内，一只手拿着这张九宫格照片，每个宫格都是不同的动作和表情，有高兴的也有悲伤的，凸显出他的年轻活泼和搞怪爱玩

> **技巧**："一只手拿着这张九宫格照片"创造递归趣味。"不同动作和表情"确保每格有变化。
