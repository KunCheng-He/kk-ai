# 思维模型库 (Thinking Models Reference)

在为用户规划结构时，请参考以下模型，寻找最适合当前素材的组织方式。

## 1. 逻辑推演类
*   **金字塔原理 (Pyramid Principle)**
    *   结构：结论先行 → 以上统下 → 归类分组 → 逻辑递进。
    *   适用：严谨的说明文、汇报材料。
*   **黄金圈法则 (Golden Circle)**
    *   结构：Why (信念/目的) → How (方法/过程) → What (结果/现象)。
    *   适用：介绍新概念、愿景驱动的叙事。
*   **第一性原理 (First Principles)**
    *   结构：解构基本事实 → 重组 → 创新结论。
    *   适用：深度解析技术原理、打破常规的论证。

## 2. 叙事叙述类
*   **英雄之旅 (Hero's Journey)**
    *   结构：平凡世界 → 冒险召唤 → 跨越门槛 → 试炼 → 归来。
    *   适用：案例分析、人物传记、创业故事。
*   **SCQA 模型**
    *   结构：Situation (情境) → Complication (冲突) → Question (问题) → Answer (答案)。
    *   适用：引人入胜的开篇、问题解决型文章。

## 3. 辩证分析类
*   **正反合 (Thesis-Antithesis-Synthesis)**
    *   结构：提出观点 → 反面质疑 → 综合升华。
    *   适用：争议性话题、多角度分析。
*   **时间/空间维度**
    *   结构：过去-现在-未来；宏观-中观-微观。
    *   适用：全景式综述。
