# 欢迎使用秀米风格主题

这是一个展示 **xiumi-style** 主题和 SVG 贴图功能的示例文档。

## 特点介绍

这个主题包含丰富的装饰元素：

- :star: **星星装饰** - 用于强调重点
- :heart: **爱心图标** - 表达喜爱
- :flower: **花朵贴图** - 顶部自动装饰
- :leaf: **叶子分隔** - 章节间自动添加

## 装饰效果

正文内容会自动包裹在卡片容器中，顶部有花朵装饰，底部有叶子分隔线。

> 引用块也有特殊的样式设计，让内容更加突出。

## 代码示例

```python
# 代码块保持简洁风格
def hello():
    print("Hello, WeChat!")
```
